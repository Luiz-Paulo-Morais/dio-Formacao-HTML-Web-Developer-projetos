O que é HTML e o que são tags?

<= Voltar

O HTML (HyperText Markup Language) é a linguagem base para criar páginas web. Ele estrutura o conteúdo, como textos, imagens, links e muito mais.

As tags são os comandos usados no HTML para definir cada elemento. Elas sempre vêm entre <> e podem ter uma abertura ( <tag> ) e, muitas vezes, um fechamento ( </tag> ).

O que são atributos em HTML?

<= Voltar

Os atributos em HTML são informações adicionais incluídas nas tags para configurar ou personalizar seus elementos. Eles sempre vêm dentro da abertura da tag e seguem a estrutura:

<nome-da-tag atributo="valor">Conteúdo</nome-da-tag>

Exemplos de atributos comuns

  1. id: Identifica de forma única um elemento na página.

    <h1 id="titulo-principal">Bem-vindo ao site!</h1>

  2. class: Agrupa elementos para aplicação de estilos.

    <p class="destaque">Este é um parágrafo destacado.</p>

Exemplos de atributos específicos

  1. src (para imagens): Define a URL da imagem.

    <img src="imagem.jpg" alt="Descrição da imagem">

  2. href (para links): Especifica o destino do link.

    <a href="https://www.example.com">Visite o site</a>

  3. alt (para imagens): Fornece uma descrição alternativa caso a imagem não carregue.

    <img src="imagem.jpg" alt="Descrição alternativa">

  4. type (para botões): Define o tipo de botão em formulários.

    <button type="submit">Enviar</button>

Principais Tags

<= Voltar

Destaque no HTML com <h1>

A tag <h1> é usada para criar o título principal da página. Ele é essencial para SEO e organização do conteúdo!

Parágrafos com <p>

Use a tag <p> para criar parágrafos de texto. Simples e indispensável para organizar o conteúdo!

Realce com <mark>

A tag <mark> serve para destacar texto como se estivesse marcado com marcador amarelo.

Tamanho de texto com <small>

A tag <small> diminui o tamanho do texto. Ideal para avisos ou observações!

Itálico com <i>

Quer dar ênfase em algo? Use a tag <i> para deixar o texto em itálico!

Texto sublinhado com <u>

A tag <u> sublinha o texto, destacando-o visualmente.

Texto Forte com <strong>

Dê importância ao conteúdo usando a tag <strong>. Texto aparece em negrito e tem semântica!

Listas Ordenadas com <ol>

Crie listas numeradas com <ol> e itens com <li>.

Listas Não Ordenadas com <ul>

Para listas simples (sem ordem), use <ul> com <li>.

Links com <a>

Crie links para outras páginas usando a tag <a> e o atributo href.

Separação de Conteúdo com <hr>

A tag <hr> insere uma linha horizontal para dividir seções.

Subscrito e Sobrescrito: <sub> e <sup>

Use <sub> para subscrito e <sup> para sobrescrito.

Citações com <blockquote>

Destaque citações longas com a tag <blockquote>.
Este é um texto de citação.

Texto com <font> (obsoleto)

Apesar de desatualizado, a tag <font> altera cor, tamanho e estilo do texto.

Texto Riscado com <del>

Use a tag <del> para marcar texto excluído.

Abreviações com <abbr>

Mostre o significado de siglas com <abbr> e o atributo title.

HTML é essencial!
